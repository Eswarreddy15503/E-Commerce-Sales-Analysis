# E-Commerce-Sales-Analysis
E-Commerce Sales Analysis using Python and Power BI
# E-Commerce Sales Analysis Dashboard

## Project Overview

This project analyzes e-commerce sales data to identify:

* Top and bottom performing product categories
* Regional sales performance
* Seasonal sales trends
* Business growth opportunities

## Tools Used

* Python
* Pandas
* Matplotlib
* Power BI
* Google Colab

## Key Insights

* Health & Beauty was the highest revenue-generating category.
* São Paulo generated the highest revenue among all states.
* November 2017 recorded peak sales performance.
* Revenue showed strong growth throughout 2017 and 2018.
* Several regions and categories underperformed and require targeted strategies.

## Dashboard Features

* KPI Cards
* Monthly Revenue Trend
* Top & Bottom Categories
* Top & Bottom States
* Order Status Distribution
* Management Recommendations
